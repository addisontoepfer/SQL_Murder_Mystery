# SQL Murder Mystery - Solved!

Solved the classic Knight Lab SQL Murder Mystery using pure SQL queries in SQLite.

**Key Skills Demonstrated**
- JOINs across 9 tables (person, drivers_license, interview, facebook_event_checkin, etc.)
- Filtering with LIKE, IN, BETWEEN, date functions
- Aggregations with GROUP BY and HAVING
- Step-by-step detective reasoning
